# 🚗 CodeAlpha Car Price Prediction Model

 📌 Project Overview

This project predicts the selling price of used cars using Machine Learning. The model is built using Linear Regression after performing data preprocessing, exploratory data analysis (EDA), feature encoding, and model evaluation.

---

 📂 Dataset

- Dataset: Car Data
- Total Records: 301
- Features Used:
  - Year
  - Present Price
  - Driven_kms
  - Fuel Type
  - Selling Type
  - Transmission
  - Owner

---

 ✨ Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Encoding
- Linear Regression Model
- Model Evaluation
- Data Visualization
- Save Trained Model (.pkl)

---

 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

 📈 Model Performance

- Mean Absolute Error (MAE): **1.54**
- R² Score: **0.74**
